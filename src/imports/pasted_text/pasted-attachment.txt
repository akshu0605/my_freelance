# Portfolio Website Design Prompt for Figma AI

Create a modern, premium, conversion-focused portfolio website for a two-person design and development studio called "Akshit & Aditi Studio".

The website should NOT look like a generic agency website. It should feel like a high-end creative studio that designs and develops websites for startups, cafes, restaurants, local businesses, and growing brands.

## Design Direction

Use the attached reference layout as inspiration ONLY for:

* Section spacing
* Visual hierarchy
* Card layouts
* Alternating content blocks
* Testimonial placement
* Project showcase flow

Do NOT copy the cleaning service design.

The final design should feel like a blend of:

* Modern startup agency
* Premium café branding
* Creative portfolio showcase
* Apple-level simplicity
* Framer-style modern layouts
